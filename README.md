# gps-tracker
Это прототип GPS трекера на esp32-cam (без камеры) и gps модуле neo-6m

Схема подключения:
Esp32-cam ------> GPS (neo-6m) \n
3V3       ------> VCC\n
GND       ------> GND\n
IO13      ------> TX\n
IO12      ------> RX\n
